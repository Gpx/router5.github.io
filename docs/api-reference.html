Router5

Constructor

Router5(routes, [opts])
Params
routes RouteNode[]Object[]RouteNodeObject The router routes
opts Object={} The router options: useHash, defaultRoute and defaultParams can be specified.
Return Router5 The router instance

Static properties and methods

Router5.ERR Error codes
Router5.transitionPath An helper function to return instructions for a transition: intersection route name, route names to deactivate, route names to activate
Params
toState Object The state to go to
fromState Object The state to go from
Return Object An object containing 'intersection', 'toActivate' and 'toDeactivate' keys

Methods

router5.setOption(opt, val) Set an option value
Params
opt String The option to set
val * The option value
Return Router5 The Router5 instance
router5.setAdditionalArgs(args) Set additional arguments used in lifecycle functions. Additional arguments are used in canActivate and canDeactivate in first positions (before toState).
Params
args Array The additional arguments
router5.getAdditionalArgs() Return additional arguments used in lifecycle functions
router5.add(routes) Add route(s)
Params
routes RouteNode[]Object[]RouteNodeObject Route(s) to add
Return Router5 The Router5 instance
router5.addNode(name, path, [canActivate]) Add a route to the router.
Params
name String The route name
path String The route path
canActivate Function A function to determine if the route can be activated. It will be invoked during a transition with `toState` and `fromState` parameters.
Return Router5 The Router5 instance
router5.useMiddleware(fn) Set a transition middleware function .useMiddleware(fn1, fn2, fn3, ...)
Params
fn Function The middleware function
router5.start([startPathOrState], [done]) Start the router
Params
startPathOrState StringObject An optional start path or state (use it for universal applications)
done Function An optional callback which will be called when starting is done
Return Router5 The router instance
router5.stop() Stop the router
Return Router5 The router instance
router5.getState() Return the current state object
Return Object The current state
router5.isActive(name, [params], [strictEquality], [ignoreQueryParams]) Whether or not the given route name with specified params is active.
Params
name String The route name
params Object={} The route parameters
strictEquality Boolean=false If set to false (default), isActive will return true if the provided route name and params are descendants of the active state.
ignoreQueryParams Boolean=true Whether or not to ignore URL query parameters when comparing the two states together. query parameters when comparing two states together.
Return Boolean Whether nor not the route is active
router5.areStatesDescendants(parentState, childState) Whether two states are descendants
Params
parentState Object The parent state
childState Object The child state
Return Boolean Whether the two provided states are related
router5.registerComponent(name, component) Register an active component for a specific route segment
Params
name String The route segment full name
component Object The component instance
router5.canDeactivate(name, canDeactivate) Shortcut to "registerComponent". It updates the "canDeactivate" status of a route segment.
Params
name String The route segment full name
canDeactivate Boolean Whether the segment can be deactivated or not
Return
router5.deregisterComponent(name) Deregister an active component
Params
name String The route segment full name
Return Router5 The router instance
router5.canActivate(name, canActivate) A function to determine whether or not a segment can be activated.
Params
name String The route name to register the canActivate method for
canActivate Function The canActivate function. It should return `true`, `false` or a promise
Return Router5 The router instance
router5.buildUrl(route, params) Generates an URL from a route name and route params. The generated URL will be prefixed by hash if useHash is set to true
Params
route String The route name
params Object The route params (key-value pairs)
Return String The built URL
router5.buildPath(route, params) Build a path from a route name and route params The generated URL will be prefixed by hash if useHash is set to true
Params
route String The route name
params Object The route params (key-value pairs)
Return String The built Path
router5.buildState(route, params) Build a state object from a route name and route params
Params
route String The route name
params Object The route params (key-value pairs)
Return String The built Path
router5.matchPath(path) Match a path against the route tree.
Params
path String The path to match
Return Object The matched state object (null if no match)
router5.urlToPath(url) Parse / extract a path from an url
Params
url String The URL
Return String The extracted path
router5.matchUrl(url) Parse path from an url and match it against the route tree.
Params
url String The URL to match
Return Object The matched state object (null if no match)
