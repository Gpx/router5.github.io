Transition

It is possible to register a "middleware" function: it can return a boolean, a promise or call a done callback.

let onTransition = function (toState, fromState, done) {
    // Let's fetch data and call done
    done(null);
};

router.onTransition(onTransition)

This type of function is ideal to remove data loading logic from components, and is a good fit for applications aiming at having a centralised state object.
